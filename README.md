歩行データベース2019に関するものは.gitignoreになっています．

## 研究概要
本研究では、腰部に装着した単一のウェアラブルセンサーから取得できる加速度データを用いて、歩行時の膝関節負荷リスクを推定する機械学習モデルの構築に取り組みました。膝関節への過剰な負荷は、変形性膝関節症や歩行障害の進行要因となります。しかし従来の評価にはモーションキャプチャや床反力計などの大型設備が必要であり、日常生活で継続的に測定することは困難でした。そこで、日常生活下でも利用可能な簡易評価手法の実現を目的として研究を行いました。

## 担当した役割
研究の企画立案からデータ分析、機械学習モデル構築、性能評価までを担当しました。具体的には、300名分の歩行データを対象に、加速度データから特徴量を抽出し、特徴選択・分類モデルの比較検証を実施しました。また、膝関節の屈曲角度および膝にかかる衝撃を基に膝関節負荷の評価基準を定義し、教師データの作成も行いました。

## 直面した課題と解決方法
ウェアラブルセンサーのみでは取得できる情報が限られるため、膝関節負荷を表現できる特徴量設計が課題でした。そこで、従来の時間領域・周波数領域特徴に加え、複数の特徴抽出手法を組み合わせたマルチドメイン特徴を導入しました。さらに特徴選択を行い、性能を維持しながら特徴数を削減することで、解釈性と汎化性能の向上を図りました。

## 技術情報
- 言語：Python
- データ規模：被験者300名分の歩行データ
- 入力データ：腰部加速度センサーデータ
- 教師データ：モーションキャプチャおよび床反力計から算出した膝関節負荷指標
- 特徴量：時間領域特徴、周波数領域特徴、マルチドメイン特徴
- 特徴選択：MRMR
- 評価方法：被験者非依存検証（Leave-One-Subject-Out）
- 評価指標：ROC-AUC

## 成果
マルチドメイン特徴を用いることで従来特徴を上回る性能を達成し、ROC-AUC 0.75〜0.77で膝関節負荷リスクを推定できることを確認しました。本研究により、単一のウェアラブルセンサーのみで膝関節負荷リスクを評価できる可能性を示し、日常生活下での継続的な歩行モニタリングや早期介入への応用が期待されます。
